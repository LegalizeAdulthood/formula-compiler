method setting

This setting specifies the default drawing method (from the Formula tab
of the Layer Properties tool window) for the fractal formula. If it is
not specified, Ultra Fractal uses the Guessing drawing method. There are
three options: guessing, multipass (Multi-pass Linear) and onepass
(One-pass Linear).

This setting can only be used in the default section of fractal
formulas. When the fractal formula is selected, Ultra Fractal resets the
drawing method to the specified value.

Example:

    default:
      method = onepass
