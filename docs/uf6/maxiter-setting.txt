maxiter setting

This setting is an int value that specifies the default maximum
iterations value (from the Formula tab of the Layer Properties tool
window) for the fractal formula. If it is not specified, Ultra Fractal
uses the default value 100. The maximum iterations value should be
larger than 0.

This setting can only be used in the default section of fractal
formulas. When the fractal formula is selected, Ultra Fractal resets the
maximum iterations value to this setting.

Example:

    default:
      maxiter = 500
