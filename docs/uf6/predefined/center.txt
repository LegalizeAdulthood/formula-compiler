#center (predefined symbol)

  --------- ---------- ----------- ----------
  Type      Readable   Writeable   Constant
  complex   Always     Never       Yes
  --------- ---------- ----------- ----------

This predefined symbol returns the center as set in the Location tab in
the Layer Properties tool window.

Example:

    complex c = #center
