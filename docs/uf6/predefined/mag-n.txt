#magn (predefined symbol)

  ------- ---------- ----------- ----------
  Type    Readable   Writeable   Constant
  float   Always     Never       Yes
  ------- ---------- ----------- ----------

This predefined symbol returns the magnification as set in the Location
tab in the Layer Properties tool window.

Example:

    float x = #magn
